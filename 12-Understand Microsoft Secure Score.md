# 📊 Microsoft Secure Score – Summary

## 📌 Overview
Microsoft Secure Score is a **security posture measurement tool** that shows how secure your organisation is.

✅ Purpose:
- Measure security strength  
- Provide improvement recommendations  
- Reduce risk across cloud and identity  

---

## 🧠 Key Concept

✅ Secure Score = **Security Health Score**

- Higher score → Better security posture  
- Lower score → More risks / gaps  

---

## 🔗 Types of Scores

- Microsoft Secure Score (365) → Overall security posture  
- Cloud Secure Score → Cloud-specific risk level  

✅ Both are part of:
Exposure Management in Defender portal  

---

## 🖥 Where to Access

Step 1 > Go to Microsoft Defender portal  
Step 2 > Open Exposure Management  
Step 3 > View Secure Score dashboard  

---

## 📊 What Secure Score Shows

- Current score  
- Improvement actions  
- Trends over time  
- Comparison with similar organisations  

✅ Helps track progress  

---

## 🔍 Products Covered

Secure Score includes recommendations from:

- Microsoft Entra ID  
- Defender for Endpoint  
- Defender for Identity  
- Defender for Office 365  
- Defender for Cloud Apps  
- Exchange Online  
- SharePoint Online  
- Microsoft Teams  

✅ Also supports third-party integrations:
- Okta  
- Salesforce  
- Zoom  
- GitHub  
- ServiceNow  

---

## 🛠 Recommended Actions

✅ The **Recommended Actions tab** shows:

- Security improvements to apply  
- Risk areas to fix  
- Status of each action  

---

### 📌 Action Status Types

- To address → Not started  
- Planned → Work in progress  
- Completed → Implemented  
- Risk accepted → Accepted risk  
- Third-party → Covered externally  

---

## ⚙️ How to Improve Score

Step 1 > Review recommendations  
Step 2 > Prioritise high-impact actions  
Step 3 > Implement security controls  
Step 4 > Track progress  

✅ Examples:
- Enable MFA  
- Configure Conditional Access  
- Secure endpoints  
- Protect email  

---

## 📈 Benefits

- Visibility into security posture  
- Prioritised recommendations  
- Integration across Microsoft security tools  
- Benchmarking against peers  

---

## ✅ Summary Flow

Check Score > Review Recommendations > Apply Fixes > Improve Score  

---

## 🧠 Key Takeaways

- Secure Score measures overall security posture  
- Higher score = stronger protection  
- Covers multiple Microsoft and third-party systems  
- Helps prioritise security improvements  
- Easy way to track and improve security
