# 🚨 Microsoft Defender XDR – Configure Advanced Features (Alerts) (Summary)

## 📌 Overview
The **Advanced Features (Alert-focused settings)** allow you to configure key capabilities that enhance:

- Incident investigation  
- Threat detection  
- Response actions  

✅ Purpose:
Improve how alerts are analysed and handled  

---

## 🧠 Key Concept

✅ Advanced Alert Features = **Investigation + Response Control**

- Enable deeper device investigation  
- Control how threats are blocked  
- Enhance SOC response capabilities  

---

## 🖥 Where to Access

Step 1 > Go to https://security.microsoft.com  
Step 2 > Settings > Endpoints  
Step 3 > Advanced features  
Step 4 > Toggle features ON/OFF  
Step 5 > Click Save  

---

## 🔑 Key Alert-Focused Features

---

### 💻 Live Response

✅ Description:
- Allows remote investigation on devices  
- Uses secure shell connection  

✅ Use:
- Analyse compromised devices  
- Run commands remotely  

✅ Requirement:
- RBAC permissions  

---

### ⚠️ Live Response – Unsigned Scripts

✅ Description:
- Allows execution of unsigned scripts  

✅ Use:
- Advanced investigations  
- Custom scripts during incident response  

⚠️ Warning:
- Higher security risk  
- Enable only when needed  

---

### 🌐 Custom Network Indicators

✅ Description:
- Create allow/block rules for:
  - IP addresses  
  - Domains  
  - URLs  

✅ Use:
- Block malicious infrastructure  
- Allow trusted resources  

✅ Controlled via:
Indicator lists  

---

## ✅ Summary Flow

Enable Feature > Configure Controls > Investigate Alerts > Take Action  

---

## 🧠 Key Takeaways

- Live Response enables real-time investigation  
- Unsigned scripts increase flexibility (with risk)  
- Network indicators control external connections  
- These features improve SOC response capability  

---

## 🔐 Security Insight

👉 Control + Visibility = Response Power  

The more control you have over endpoints,  
the faster you can stop threats
