# 🧠 Microsoft Defender XDR – Threat Intelligence Briefing Agent (Summary)

## 📌 Overview
The **Security Copilot Threat Intelligence Briefing Agent** helps security teams:

- Generate threat intelligence briefings  
- Analyse recent attacker activity  
- Correlate threats with organisational risk  

✅ Purpose:
Turn complex threat data into **actionable intelligence (fast)**  

---

## 🧠 Key Concept

✅ AI-Powered Threat Briefing  

- Collects data from multiple sources  
- Analyses threats automatically  
- Produces a structured security report  

✅ Target users:
- SOC analysts  
- Security managers  
- CISOs  

---

## 🚀 Why It Matters

Without agent:
- Manual data collection  
- Multiple tools required  
- Time-consuming analysis  

With agent:
- Automated analysis  
- Faster insights  
- Prioritised risks  

---

## 📍 Where to Find It

Step 1 > Go to Microsoft Defender portal  
Step 2 > Navigate: Threat intelligence > Threat analytics  
Step 3 > View agent banner at top  

---

## ⚙️ Prerequisites

✅ Required:
- Microsoft Security Copilot enabled  
- Threat Intelligence plugins installed  

✅ Permissions:
- Security Reader (view results)  
- Security Admin (setup)  
- Access to vulnerability data  

---

## ▶️ Run a Briefing

Step 1 > Select Run agent  
Step 2 > Wait for analysis  
Step 3 > Open full briefing  

✅ Output includes:
- Active threats  
- Vulnerabilities  
- Business impact  

---

## 📊 What the Briefing Shows

- Threat actor activity  
- Organisational exposure  
- Attack trends  
- Risk prioritisation  

✅ Goal:
Help prioritise security actions  

---

## 📥 Share Results

Step 1 > Copy briefing  
OR  
Step 2 > Download as Markdown  

✅ Use for:
- Reports  
- Team updates  
- Leadership briefings  

---

## ⚙️ Manage Settings

Step 1 > Select Manage agent  
Step 2 > Configure:
- Schedule  
- Preferences  

✅ Also accessible via:
System > Settings > Defender XDR  

---

## 🔍 Review Past Activity

Step 1 > Open Security Copilot Activity  
Step 2 > View:
- Previous briefings  
- Run status  
- Reports  

---

## 🧠 View Agent Reasoning

Step 1 > Open briefing  
Step 2 > Select View activity  

✅ Shows:
- How the agent analysed data  
- Decision flow  

---

## 👍 Provide Feedback

Run Agent → Analyse Threat Data → Generate Briefing → Review Insights → Take Action
